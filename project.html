roblox studio project • 2025

This project procedurally generates chunks of terrain around the player locally, utilizing a server-based perlin noise calculation; it actively adds and removes chunks when the player moves, taking measures to reduce lag. This randomizes occurences in terrain (i.e. trees, water) based on y height.

This project is a custom class selection menu that allows the user to choose a class via visible rigs.

The screen shifts towards where the mouse moves, and the rigs highlight when the mouse hovers over them. In additon to this, there is a tweened popup gui that displays information on the class and allows the selection of the class.

I also worked on custom abilities for each class. This mainly involves unique systems to cast/use abilities and vfx.
